Built-in Methods

Maak een eigen voorbeeld van de volgende functies en licht toe wat de functie doet in het blok:

  1. charAt()
  2. conCat()
  3. slice()

charAt()

let sentance = 'abcdefg'
console.log(sentance.charAt(4)); "E"

conCat()

let str1 = 'hello';
let str2 = 'world';
console.log(str1.conCat(" ", str2)); "hello world"

slice()

let str = "abcdefg";
console.log(str.slice(0, 3)); "abc"

split()

let str = "abc def ghi jkl mno";
console.log(str.split(" ")); array[ "abc", "def", "ghi", "jkl", "mno"]

substr()

let str = "Hello";
console.log(str.substr(1)); llo
