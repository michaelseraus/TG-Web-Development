Data types herkennen

1) Hoe herken je dat een waarde String is? Geef ook een voorbeeld

a String must be surrounded by quotes e.g : let string = " This is a string";

2) Hoe herken je dat een waarde Number is? Geef ook een voorbeeld.

By it being a numeric sequence e.g: let number = 56;

3) Hoe herken je dat een waarde BigInt is? Geef ook een voorbeeld.

If there is an N to the end of an integer e.g: let bigInt = 1234567892345532n ;

4) Hoe herken je dat een waarde Symbol is? Geef ook een voorbeeld.

You recognise it by it being a built in global function e.g : let mySymbl = symbol()
