Numbers

1) Copy de volgende code in jouw VSCode:

let number = 3 ;
number ++; 2.

2) Print uitkomst in jouw browser console. 4

3. Wijzig number++ naar number--.
4. Wat is de uitkomst van de som ? 2

5. Copy de volgende code in jouw VSCode:

let number = 3 ;
const sum = number++ +2 ;
console.log(sum);

6. Wat is de uitkomst van van de som ? 5 cause the increment operater was post operand.

7) Copy de volgende code in jouw VSCode:

let number = 3 ;
const sum = ++number +2 ;
console.log(sum);

8. Wat is de uitkomst van van de som ? 6 cause the increment operator was pre operand.
