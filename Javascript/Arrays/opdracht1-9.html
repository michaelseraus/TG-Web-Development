Opdracht 6:

Zoek uit wat push(), pop(), shift(), slice(), splice(), sort().
Maak gebruik van de fruits array en maak voor elke functie
een voorbeeld en geef in commentaar wat het doet.

push()

The push() method adds one or more elements to the end of an array and returns the new length of the array.

const fruits = ["banana", "apple", "kiwi", "orange", "aardbei"];
const fruitLength = fruits.push("watermelon");
console.log(fruits); Logs the fruit array including the newly "pushed" value 'watermelon'.
console.log(fruitLength) Returns the new Array length of fruits which would be 6;

pop()

The pop() method removes the last element from an array and returns that element.
This method changes the length of the array.

const fruits = ["banana", "apple", "kiwi", "orange", "aardbei"];
const lastElement = fruits.pop();
console.log(fruits); Logs the fruit array minus the last element.
console.log(lastElement); Logs the popped fruit "aardbei"

shift()

The shift() method removes the first element from an array and returns that removed element.
This method changes the length of the array.

const fruits = ["banana", "apple", "kiwi", "orange", "aardbei"];
const firstElement = fruits.shift();
console.log(fruits); logs the fruit array minus the first element.
console.log(firstElement); Logs the first element that was taken out of the array : 'banana'.

slice()

The slice() method returns a shallow copy of a portion of an array into a new array object
selected from start to end (end not included) where start and end represent
the index of items in that array. The original array will not be modified.

const fruits = ["banana", "apple", "kiwi", "orange", "aardbei"];
console.log(fruits.slice(1)); Logs from index [1] till end of array :["apple", "kiwi", "orange", "aardbei"]
console.log(fruits.slice(1, 4)); Log from index[1] to [3] ( end not included):['apple','kiwi','orange']

splice()

The splice() method changes the contents of an array by removing or replacing existing elements and/or adding new elements in place.

const fruits = ["banana", "apple", "kiwi", "orange", "aardbei"];
fruits.splice(1, 0, "berries"); Added "berries" on index[1] and didnt delete anything.
console.log(fruits); ["banana","berries" "apple", "kiwi", "orange", "aardbei"]
fruits.splice(1, 2, "apple"); Added "apple" on index[1] and deleted 2 elements starting from index[1]
console.log(fruits);["banana", "apple", "kiwi", "orange", "aardbei"]

sort()

The sort() method sorts the elements of an array in place and returns the sorted array.
The default sort order is ascending, either numerically or alpabetically by comparing their sequences.

const fruits = ["banana", "apple", "kiwi", "orange", "aardbei"];
console.log(fruits.sort()); ['aardbei', 'apple', 'banana','kiwi' 'orange']
const numbers = [12, 123, 4, 2, 35, 56];
console.log(numbers.sort()); [12, 123, 2, 35, 4, 56]
