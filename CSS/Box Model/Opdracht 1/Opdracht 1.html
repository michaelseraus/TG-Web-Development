Wat zijn de eigenschappen van block boxes en inline boxes?

Block box

Na block boxes heb je altijd een line break, dus block boxes kunnen niet naast elkaar staan.
Ook maakt block boxes het mogelijk om de hoogte en breedte van de box te bepalen.

Inline Boxes

Bij een inline box krijg je geen line break, dus kunnen ze naast elkaar staan.
verder kan je ook geen breedte of hoogte bepalen.
