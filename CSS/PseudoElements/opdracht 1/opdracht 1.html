Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)?
Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

::before

Inserts content before the selected element which can then be styled.

::after

Inserts content after the selected element which can then be styled.

::first-line

Used to select the first line of a text to be styled

::first-letter

Selects the first letter of an element to be styled.
