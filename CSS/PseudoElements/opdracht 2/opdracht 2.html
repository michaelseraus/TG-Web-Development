Opdracht 2 Er zijn 14 verschillende pseudo elementen. Je hebt in de voorgaande opdracht
reeds 4 kunnen benoemen.
Kun je de andere 10 ook benoemen en aangeven wat ze precies doen?

::backdrop

renders a box the size of the viewport beneath any element being presented in fullscreen-mode

::cue

Is used to style captions and other cues with video text tracks

::cue-region

Is used to style captions of a specific region

Syntax=  ::cue-region( selector )

::grammar-error

Is used to style grammatical errors that the user agent has flagged

::marker

Selects the marker box of a list item
so you can style the bullets/numbers

::part()

Selects any elemets that has a matching part attribute

Syntax=   ::part(partIdName)

::placeholder

Selects the placeholder text in an input or textarea

::selection

Used to apply styles to selected areas in a document by the user with mouse

::slotted()

Selects any element placed in a slot inside a html page

::spelling-error

Used to style a text segment which the user agent has flagged as incorrectly spelled
