Er zijn verschillende Selectors bijvoorbeeld :
type Selectors
Descendant Selectors
Class Selectors
Geef voorbeelden van elk type selectors en geef aan hun voordelen

Type selectors : h1, div, header, table, footer,
section voordeel ervan is dat je heel makkelijk een html element kan selecteren.

Descendant Selectors : div p , header h1
je kan alle descendants van een element met dezelfde naam selecteren.

Class selectors : .myClassName
hiermee kan je classes selecteren die je zelf een naam had gegeven.
