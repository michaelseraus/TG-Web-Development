Verder heb je ook nog de volgende selectors:
● child selectors
● adjacent selectors
● general selectors
Maak in een aparte html bestand voor elke selector een voorbeeld.

Child Selector

Selecteert alle elementen die "children" zijn van een specifieke element.
Bijv : div > p Link to example

Adjacent Selectors

Selecteert een element die direct na een specifieke element komt
bijv : div + p Link to Example

General Sibling Selector

Selecteert elementen die "siblings" zijn van een bepaald element.
bijv : div ~ p Link to Example
