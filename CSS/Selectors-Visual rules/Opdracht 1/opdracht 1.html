Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Je kan op 3 verschillende manieren CSS inladen :

inline CSS >>>> Dit wordt gedefiniert in de "style" attribuut van een element Bijvoorbeeld :

Example inline CSS

External CSS >>>> dit word gedefiniert in the "link" tag in je HTML. In essentie link je naar essentie externe .css file waar all je styling in zit

Internal CSS >>>> Dit wordt in de "style" tag gedefiniert in je HTML bestand.
