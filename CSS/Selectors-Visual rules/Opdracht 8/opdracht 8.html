Leg in eigen woorden uit wat met cascade en inheritance wordt bedoeld. Maak voorbeelden om je antwoord uit te leggen.

Cascade

This means that if you have 2 rules that can apply to an element, the last rule in the source order will "win" Example link

inheritance

this means that if you have certain rules applied to an element, that it will also be applied to each element inside it UNLESS you directly apply different rules for the specific element.
